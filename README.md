# Javier Alaves

This repository contains the source code for the personal website of **Javier Alaves**, a Staff Product Designer bringing the world to new standards. The site is built with Next.js and Tailwind CSS.

## Running locally

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then visit [http://localhost:3000](http://localhost:3000) in your browser.
